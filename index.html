Quick Setup Guide

Get Started in Minutes

1. Clone Repository
git clone https://github.com/yourusername/nlp-mini-project.git cd nlp-mini-project
2. Install Dependencies
pip install -r requirements.txt
3. Download Dataset

Download the Sentiment140 dataset and place it as sentiment140.csv in the project root.

Download Dataset
4. Run Application
python app.py

Then visit http://localhost:5000
