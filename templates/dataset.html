Dataset Description

About Sentiment140

The Sentiment140 dataset contains 1.6 million labeled tweets with positive and negative sentiments. It's widely used for sentiment analysis research and benchmarking.

The dataset was created by Stanford University and includes tweets from various topics, making it ideal for training robust sentiment analysis models.

Data Processing Pipeline
  • Text Cleaning: URLs, mentions, hashtags removed
  • Tokenization: Text split into individual words
  • Stopword Removal: Common words filtered out
  • Lemmatization: Words reduced to root form
  • Vectorization: TF-IDF feature extraction
Key Insight: This dataset contains labeled tweets used for training the ML model. The model learns patterns from this data to predict sentiment for new, unseen tweets with high accuracy.
